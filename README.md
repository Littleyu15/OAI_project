# Repository File Guide

## 1️⃣`Lab_handover/`

### 目的
記錄每個學長交接的實驗室工作內容。

### 應該包含
- 交接的工作內容
- 後續需要做的事
- 尚未確認的問題

### 完成標準
之後回頭看這份文件，可以知道交接了什麼、需要負責什麼、卡住時要問誰。

## 2️⃣`Daily_logs/`

### 目的
記錄每日進度。

### 應該包含
- 日期
- 完成事項
- 相關文件連結

### 完成標準
可以了解每日具體做了哪些工作項目。

## 3️⃣`OAI/`

### 目的
紀錄Richard實驗驗證、OAI相關背景知識、HTC attack記錄

### 應該包含
- 安裝流程
- 驗證流程
- 測試結果
- 原理特性介紹

### 完成標準
能跑出與Richard論文相對應的結果，以及能夠理解原理特性。
