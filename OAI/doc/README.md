## OAI 5GC-gNB-nrUE Architecture

當一個 5G 設備要求上網時，`AMF` 處理連線，`AUSF`/`UDM`/`UDR`/`MySQL` 確認身分，`SMF` 建立上網通道並發配 IP，最後由 `UPF` 負責將設備的資料轉到 `Ext-DN` 完成上網。
```
                           +----------------------+
                           | Internet(oai-ext-dn) |
                           +----------------------+
                                      ^
                                      |
                                      |
                        +--------------------------+
                        |           UPF            |
                        |  User Plane Function     |
                        +--------------------------+
                                      ^
                                      |
                                      |
        +---------------------------------------------------+
        |                     OAI 5GC                       |
        |                                                   |
        |  +--------+   +--------+   +--------+             |
        |  |  AMF   |   |  SMF   |   |  NRF   |             |
        |  +--------+   +--------+   +--------+             |
        |       |             |                             |
        |  +--------+   +--------+                          |
        |  | AUSF   |   |  UDM   |                          |
        |  +--------+   +--------+                          |
        |                                                   |
        |      (Docker Compose Containers)                  |
        +---------------------------------------------------+
                                      ^
                                      |
                                      |
                                      v

+---------------------------------------------------------------+
|                           OAI gNB                             |
|                                                               |
|  +---------------------------------------------------------+  |
|  |                    nr-softmodem                         |  |
|  |                                                         |  |
|  |   RRC / PDCP / RLC / MAC / PHY (only do transfer)       |  |
|  |                                                         |  |
|  |   Band : n78                                            |  |
|  |   PRB  : 106 (40MHz)                                    |  |
|  |   SCS  : 30kHz                                          |  |
|  +---------------------------------------------------------+  |
|                                                               |
|                 RF Simulator (--rfsim)                        |
+---------------------------------------------------------------+
                                      ^
                                      |
                                      | RF sim
                                      v

+--------------------------------------------------------------+
|                          OAI nrUE                            |
|                                                              |
|  +--------------------------------------------------------+  |
|  |                    nr-uesoftmodem                      |  |
|  |                                                        |  |
|  | NAS / RRC / PDCP / RLC / MAC / PHY                     |  |
|  |                                                        |  |
|  | IMSI : 001010000000001                                 |  |
|  | Band : n78                                             |  |
|  | 40MHz / 30kHz SCS                                      |  |
|  +---------------------------------------------------------+ |
|                                                              |
|         RF Simulator Client (127.0.0.1)                      |
+--------------------------------------------------------------+
                                ^
                                | (IP Packets / Ping / iPerf)
                                |
+--------------------------------------------------------------+
|                          UE Host                             |
|                 (Application / Traffic Generator)            |
+--------------------------------------------------------------+

```
